vimplenote-vim
==============


Description
-----------
Simplenote client for vim written in pure vimscript.

Requires
--------
* [curl command](http://curl.haxx.se/) : http://curl.haxx.se/
* [webapi-vim](https://github.com/mattn/webapi-vim) : https://github.com/mattn/webapi-vim
